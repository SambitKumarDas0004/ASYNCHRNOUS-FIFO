# ASYNCHRNOUS-FIFO
This project implements an asynchronous FIFO that safely transfers data between two independent clock domains (write clock and read clock). It uses Gray-code pointers and double-flip-flop synchronization to avoid metastability and ensure reliable full/empty detection.
